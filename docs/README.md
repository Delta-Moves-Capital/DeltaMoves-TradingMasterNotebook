## Conventions for Adding New Documents
- Use lowercase file names with hyphens (e.g., `spx-iron-condor-rotation.md`).
- Place all general-purpose documents in `/docs/`.
- Subdirectories may be used for large categories (e.g., `/docs/trading/`, `/docs/options/`).
- Include a short description at the top of each document.
